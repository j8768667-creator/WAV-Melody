Analysis Settings Help

Use the narrowest practical note range, then adjust confidence and timing controls.

Minimum and maximum note

These define the pitch range the analyser may accept. A narrower range reduces octave mistakes and false harmonics. Typical ranges: bass 28–55, male vocal 40–72, female vocal 48–84, guitar melody 40–84, general chords 36–84.

Minimum confidence

This sets how certain the analyser must be. Lower values recover more quiet or unclear notes but create more wrong notes. Higher values produce cleaner MIDI but may leave gaps. Start around 55–65%. Raise it for false notes or unstable chord names; lower it when genuine notes are missing.

Minimum note length

Mainly affects melody analysis. Notes shorter than this number of milliseconds are removed. Use 40–80 ms for fast leads, 80–150 ms for normal melodies, 150–250 ms for vocals, and 100–250 ms for bass. Raise it to remove tiny glitches; lower it when real short notes disappear.

Chord change hold

A chord must remain stable for this long before it is accepted. Use 150–300 ms for fast stabs, 250–500 ms for pop or rock rhythm, 350–600 ms for piano chords, and 700–1200 ms for slow pads. Raise it to stop rapid false changes; lower it when genuine quick changes are missed.

Recommended adjustment order

Melody: note range → frame size → confidence → minimum note length.
Chords: note range → frame size → confidence → chord change hold.

Chord recognition currently targets major and minor triads. Seventh, suspended, diminished and slash chords may be simplified to the nearest major or minor chord.
